About
=====

This vagrant environment sets up vanilla Ubuntu 12.04 and adds the required set
of PPAs to install plainbox. The plainbox package is a development tool and
swiss-army-knife for the new core of checkbox, known as plainbox

Prerequisites
=============

You will need a working Vagrant installation.

Usage
=====

Run 'vagrant up' and wait for the virtual machine to provision. Then run
'vagrant ssh' and you should be able to use 'plainbox'
